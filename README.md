## 🌟 Overview

**ProTrack Ultimate** is not just another Pomodoro timer. It is a comprehensive, desktop-class web application designed from the ground up to hyper-optimize your focus and workflow. Built entirely with **Vanilla JavaScript, HTML5, and CSS3 (Zero Dependencies)**, it boasts a blazing-fast performance, offline capabilities, and a highly polished glassmorphism UI.

Whether you're a developer needing deep work sessions, a student tracking study hours, or anyone looking to manage tasks effectively, ProTrack adapts to your workflow seamlessly.

---

## ✨ Ultra-Premium Features

### ⏱️ Advanced Pomodoro Engine
- **Picture-in-Picture (PiP) Mode**: Keep the timer floating above all your other applications. Never lose track of time when switching tabs.
- **Zen Mode**: A distraction-free, full-screen immersive mode that hides everything except your timer.
- **Ambient Soundscapes**: Built-in Lo-Fi cafe and rain sounds to get you into the zone.
- **Dynamic Title & Notifications**: Browser tab updates and native push notifications when sessions end.

### 📋 Intelligent Task Management
- **Kanban-style Drag & Drop**: Intuitively organize tasks into 'To Do', 'In Progress', and 'Completed'.
- **Subtasks & Progress Tracking**: Break down complex tasks into manageable subtasks.
- **Pomodoro Linking**: Assign and track the exact number of Pomodoros spent on specific tasks.

### 📊 Comprehensive Analytics
- **GitHub-style Heatmap**: Visualize your productivity streak and daily focus intensity over a 365-day grid.
- **Real-time Statistics**: Interactive charts tracking your focus hours by category (Work, Study, Personal).
- **Journal History**: Auto-generated log of your completed sessions and milestones.

### 🎨 Personalization & UX
- **Theme Engine**: Switch between multiple curated themes (Cyberpunk Neon, Matcha Green, Default Glass).
- **PWA Ready**: Installable on Desktop, iOS, and Android. Works **100% offline**.
- **Virtual Focus Tree**: Gamified progress where a virtual tree grows as you maintain your focus streak.

---

## 🛠️ Tech Stack & Architecture

This project was intentionally built without heavy frameworks to demonstrate mastery over the core web technologies:

* **Frontend**: HTML5, CSS3 (CSS Variables, Flexbox, Grid, Glassmorphism, Animations)
* **Logic**: Vanilla JavaScript (ES6+, DOM Manipulation, Drag & Drop API, Picture-in-Picture API)
* **Storage**: LocalStorage API for persistent data without a database.
* **Service Worker**: Caching strategies for complete offline Progressive Web App (PWA) support.
* **Charting**: Chart.js (The only external library used for data visualization).

## 🚀 Getting Started

### Prerequisites
Since this is a client-side application, you only need a modern web browser (Chrome, Edge, Firefox, Safari).

### Local Setup
1. Clone the repository:
   ```bash
   git clone https://github.com/Triargi/productivity-tracker.git
   ```
2. Navigate into the directory:
   ```bash
   cd productivity-tracker
   ```
3. Open `index.html` in your browser, or serve it using a local server (e.g., Live Server or Python's `http.server`):
   ```bash
   npx serve .
   ```

---

## 📱 PWA Installation

1. Open the [Live Demo](https://productivity-tracker.muhammadtriargi.workers.dev/) in Google Chrome or Microsoft Edge.
2. Click the **Install App** icon located in the URL address bar.
3. ProTrack will now appear as a native application on your device!
